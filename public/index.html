api تقویم ایران

به همراه وقایع شمسی

سلام دنیا

به api تقویم فارسی ایران خوش‌ آمدید این api برای ساده سازی کار برنامه نویسان به وجود آمده است و اصولا در پروژه هایی که نیاز به یک ساعت و تاریخ دقیق از سمت سرور دارند میتواند استفاده شود همچنین این api از یک سیستم نمایش رویداد های شمسی ایران نیاز پشتیبانی میکند :) کد های این پروژه را میتوانید از گیت هاب دانلود کنید

چطوری کار میکنه ؟‌

زمان ‌

GET /time
برای مثال : http://api.felfelpardaz.ir/time

نتیجه

2:55:21 AM

تاریخ فعلی ‌

GET /date
برای مثال : http://api.felfelpardaz.ir/date

نتیجه

یک json بر میگرداند که حاوی مقادیر زیر است
date : تاریخ فعلی به میلادی
jdate : تاریخ فعلی به شمسی
event : رویداد مربوط به امروز
week : نام روز هفته

شمسی - تاریخ درخواستی ‌

GET /date/1395-01-01 برای مثال : http://api.felfelpardaz.ir/date/1395-01-01

نتیجه

یک json بر میگرداند که حاوی مقادیر زیر است
date : تاریخ درخواستی به میلادی
jdate : تاریخ درخواست به شمسی
event : رویداد مربوط به روز
week : نام روز هفته

تبدیل

این متد برای تبدیل تاریخ استفاده میشود و دو ارگومان میگیرد ارگومان اول نوع تبدیل و آرگومان دوم تاریخ مربوطه

GET to/[type]/[date] 
 ارگومان type دو حالت میتواند باشد 
  
jalali (شمسی)
 این نوع  برای تبدیل تاریخ میلادی به شمسی استفاده میشود 

Gregorian (میلادی) 
 این نوع جهت تبدیل به میلادی مورد استفاده میگیرد

برای مثال
 http://api.felfelpardaz.ir/to/jalali/2013-08-02
نتایج را مانند متد date بر میگرداند
